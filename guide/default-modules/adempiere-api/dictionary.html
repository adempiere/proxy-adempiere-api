# Dictionary Service

# GET /api/dictionary/window

Gets the metadata of the window, the metadata of the tabs it contains and the metadata of the list of fields.

# Parameters:

  • token - user token returned from POST /adempiere-api/user/login.
  • language - language for response translations.
  • id - window identifier.
  • uid - universally unique window identifier.

# Body of the Petitión:

null

# Example of a Call:

curl --silent --location --request GET 'https://api.erpya.com/api/adempiere/dictionary/window?token=%3Ctoken-generated-for-demo-api%3E&language=es&uuid=a52192bc-fb40-11e8-a479-7a0060f0aa01&id=53102'
