# Paràmetros POST:

container_uuid - UUID del contenedor, normalmente es una ventana column_name - nombre de atributo o columna is_for_current_user - se aplica al usuario actual (booleano) is_for_current_client - se aplica al cliente actual (booleano) is_for_current_organization - se aplica a la organización actual (booleano) is_for_current_container - se aplica al contenedor actual (booleano) value - valor para establecer en la preferencia
